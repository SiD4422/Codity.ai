# Codity.ai Job Scheduler

This repository contains the submission for the Codity.ai Job Scheduler project.

## Repository Structure

- `job-scheduler/` - The main source code directory.
  - `backend/` - Node.js API server and PostgreSQL migration scripts.
  - `frontend/` - React (Vite) frontend dashboard.
  - `docs/` - Architecture diagrams, API specifications, and design decisions.
  - `README.md` - Detailed instructions on setting up and running the project.
- `Codity_Job_Scheduler_Submission.docx` - Written submission document.
- `Codity_Job_Scheduler_Submission.pdf` - Written submission document (PDF).

## Getting Started

Please refer to the [Main Project README](job-scheduler/README.md) for comprehensive instructions on how to set up the database, start the API server, run the frontend, and spawn workers.
